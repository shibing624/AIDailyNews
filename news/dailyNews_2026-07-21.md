### [完全开源的 AI 工程技术课程指南](https://github.com/rohitg00/ai-engineering-from-scratch)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-07-21 00:24:48

这个开源课程旨在解决学习 AI 工程中的碎片化问题，通过 20 个阶段、503 节课和约 320 小时的学习内容，从线性代数到自主群体技术，全面覆盖 AI 技术的理论基础及实际应用。课程通过多语言编程（Python、TypeScript、Rust、Julia），以手动构建每个 AI 算法，让学习者充分理解其背后的数学原理和实现方法。课程不提供简化的代码“复制粘贴”，而注重算法从零开始的推导和实现，每节课都附带可重复使用的功能工具。此外课程提供动态学习路径，并为完成后提供可部署的工具，适合真正想深入理解 AI 技术的开发者或学生，是一个注重实战的优质学习资源。

### [Cognee：开放源码 AI 长期记忆平台](https://github.com/topoteretes/cognee)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-07-21 00:24:48

Cognee 是一个开源的 AI 记忆管理平台，为 AI 助手和智能体提供跨会话的长期记忆支持。核心功能包括数据统一摄取、自主构建知识图谱，以及基于认知科学的本体生成，使数据具有语义可搜索性和关系的动态更新。支持 Python、Rust、TypeScript 等语言，并能通过插件与热门 AI 助手（如 Claude、Cursor 等）集成。此平台支持 Postgres 后端，简化了传统记忆管理所需的多服务结构，能够有效整合图、向量、元数据和会话。适合企业统一知识库建设和多智能体协作，是构建可靠 AI 系统的优质工具。

### [OpenCut：开源跨平台视频编辑工具](https://github.com/OpenCut-app/OpenCut)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-07-21 00:24:52

OpenCut 是一个免费开源的视频编辑器，支持Web、桌面和移动设备，基于Rust核心开发。目前正在进行完全重写，新版本将包括编辑器API、插件支持、多平台适配、AI代理服务器支持及无界面自动化模式等功能。除了经典版本外，新版本仍在开发中并提供动态脚本支持。OpenCut由开源社区支持，是面向开发者和创作者的理想工具，适合自动化视频编辑、批量渲染以及高效插件开发。使用者可以持续关注其Discord社区以获得开发更新和支持。
