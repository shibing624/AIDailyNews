### [VoxCPM: 语境感知语音合成与逼真语音克隆技术](https://github.com/OpenBMB/VoxCPM)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-01-19 23:24:31

VoxCPM是一种创新的无标记分词文本到语音(TTS)系统，直接从文本生成连续语音，克服以离散方式标记语音的限制。VoxCPM的核心功能包括通过上下文感知生成具有表达性的语音，以及通过零样本语音克隆实现逼真声音再现。在NVIDIA RTX 4090 GPU上，它以高效的实时因子实现流式合成。这款模型通过支撑的低等级和LoRA微调使用户能够创建定制版语音。另外，VoxCPM也应用于双语数据（中文和英文），但在其他语言上表现不佳。
