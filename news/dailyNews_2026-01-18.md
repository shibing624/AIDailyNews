### [去中心化身份验证：Vouch 协议的崛起](https://news.ycombinator.com/item?id=46668263)

来源：Hacker News - Newest: "AI"

发布时间：2026-01-18 22:56:52

Vouch 协议是一种新的开源标准，用于替代传统的数字身份认证方式。通过使用 W3C 去中心化标识符（DID），Vouch 协议使用户的代理能够生成自身的密钥对，并将公钥发布到用户域名上，形成信任根。每个操作通过 JWT-VC 签名验证，而无需经过中心服务器。这样的设计提供了一种无需高昂费用即可实现数十亿自主代理身份验证的可扩展方法。Vouch 协议现已提交给内容出处与真实性联盟，作为与 Adobe 和 Microsoft 合作推动此去中心化模型的标准。

### [VoxCPM：真实感语音合成的新纪元](https://github.com/OpenBMB/VoxCPM)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-01-18 23:19:02

VoxCPM 是一个创新的无符号化文本转语音（TTS）系统，旨在重新定义语音合成的真实感。通过连续性空间建模，VoxCPM 克服了离散化挑战，实现了上下文敏感语音生成及真实感零样本声音克隆功能。它使用端到端扩散自回归架构直接从文本生成连续语音表现，大幅提高了表达能力和生成稳定性。VoxCPM 可以基于内容自适应调整讲话风格，支持高效率实时合成，适合多种实时应用场景。生成的语音不仅捕捉到说话者的音色，还包括细腻的特征如口音及情绪基调。
