### [Heretic：全自动解除语言模型的审查工具](https://github.com/p-e-w/heretic)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-02-23 23:43:14

Heretic 是一款工具，旨在去除基于 Transformer 的语言模型中的审查机制（又称“安全调整”），而无需昂贵的后期训练。其结合了先进的方向消融实现和基于 Optuna 的 TPE 参数优化器，实现了全自动化操作。Heretic 通过协同最小化拒绝数和模型的 KL 散度，找到了高质量的消融参数。这种方法得到的去审查模型最大程度上保留了原始模型的智能。即使用户不了解 Transformer 的内部结构，也可以通过命令行程序使用 Heretic 来为模型去审查。Heretic 支持大多数密集模型，但尚不支持 SSM/混合模型和某些新型注意力系统。

### [Zvec：嵌入式高效向量数据库](https://github.com/alibaba/zvec)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-02-23 23:43:14

Zvec 是一个开源的、可嵌入的向量数据库，旨在实现低延迟、可扩展的相似性搜索，其基于 Alibaba 的 Proxima 向量搜索引擎构建，能够处理大规模的生产环境数据。在毫秒内搜索数十亿的向量。Zvec 非常简单易用，无需服务器配置即可开始使用，并支持稠密和稀疏向量，还可以将语义相似性与结构化过滤相结合。Zvec 适用于多种平台，包括各类服务器、笔记本、命令行工具，甚至是边缘设备。同时，Zvec 允许开发者通过 Python 和 Node.js 简单地进行安装和构建。
