### [SlopGuard：助开发者识别AI生成的无效PR与问题](https://slopguard.app)

来源：Hacker News - Newest: "AI"

发布时间：2026-06-11 01:35:21

SlopGuard 是一款专为 GitHub 维护者打造的工具，旨在帮助过滤 AI 生成的低质量 PR 和问题。它通过为每个贡献打分（0-100 的“slop评分”）和提供具体标注、审查原因，让维护者能够快速辨别内容真伪。用户可以设定评分阈值，将高于阈值的内容加注标签和评论，而不会自动关闭任何内容，最终决策权仍交给人类。此外，SlopGuard 不使用 LLM，而是依赖启发式规则，确保识别高精度且避免误判。它能够帮助维护者节省时间，专注于高质量的真正贡献，是一项尊重贡献者同时提升效率的创新工具。

### [整合社交信号的AI搜索引擎「last30days」](https://github.com/mvanhorn/last30days-skill)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-06-11 01:58:07

「last30days」是一款由AI代理驱动的搜索引擎，能够实时汇总Reddit、X (Twitter)、GitHub等平台的最新信号，依据真实的用户交互（如点赞、评论、金钱交易等）对结果进行评分，从而提供关于某主题的30天最新洞察，而非传统编辑筛选内容。特点包括：支持多种社交平台并行搜索、无需复杂配置、生成简洁的HTML摘要、多设备支持等。其最大的亮点是，通过突破单一平台的限制，整合多个平台信号形成更可靠的趋势洞察，解决Google等搜索引擎无法深度涉及时效性社交信息的难题。适合关注实时热点、新闻事件和行业趋势等场景使用。

### [OpenMed：全设备运行的医疗AI工具](https://github.com/maziyarpanahi/openmed)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-06-11 01:58:07

OpenMed是一款无需云支持的本地化医疗AI工具，提供疾病诊断信息提取、PII（个人身份信息）去标化处理等功能，适用于各种硬件环境（包括苹果设备的MLX加速）。工具集成了1000多个专业医疗模型，涵盖12种语言，支持医疗文本结构化分析与隐私保护。特色功能包括一键部署、批量处理、离线运行等，使数据隐私更可控，避免患者数据离开网络。其对医疗行业意义重大，有助于提高临床数据处理效率，同时减少对第三方云服务和供应商锁定的依赖。

### [社交与技术综合搜索的全能助手：last30days](https://github.com/mvanhorn/last30days-skill)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-06-11 01:58:13

/last30days 是一个基于人工智能代理的多平台综合搜索引擎，采用用户点赞、评论及真实货币交易数据来评分，区别于传统编辑聚合方式。其特点是对 Reddit、X（Twitter）、YouTube、TikTok 等社交平台及 GitHub 等技术社区进行实时信息的爬取和搜索，生成高度相关的简短概括。通过跨平台的 API 和用户自定义的认证密钥，这款搜索工具能够整合独立的信息孤岛，提供无法在 Google 搜索中获得的最新动态，如企业动态、会议预热及工具比较等，尤其适合对快速变化的信息具有高敏感需求的领域。v3 版本还新增了 HTML 报告生成功能以及更友好的内部架构设计，便于团队分享与协作，是一个颇具创新价值的个性化搜索工具。

### [隐私优先的开源研究整理工具：Open Notebook](https://github.com/lfnovo/open-notebook)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-06-11 01:58:13

Open Notebook 是 Google Notebook LM 的开源替代方案，支持多模式、多模型的开放式研究笔记管理工具。其特色在于支持用户完全掌控数据的隐私，支持 18+ 人工智能模型提供商（包括 OpenAI、Anthropic 等），并能高效管理多种类型的内容（PDF、视频、音频等）及进行智能化的搜索与上下文关联聊天。此外，其多语言界面、多发言人播客生成、REST API 支持及自定义部署（Docker 或本地）增加了工具的可拓展性，尤其适合重视隐私和希望跨平台运用研究内容的用户。同时，该项目凭借其开放源代码和对成本的可控性，已成为学术研究与开发人员的热门选择。
