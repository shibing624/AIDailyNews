### [CloakBrowser：避免机器人检测的隐形浏览器](https://github.com/CloakHQ/CloakBrowser)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-05-09 00:14:09

CloakBrowser 是基于 Chromium 定制的隐形浏览器，专注于通过修改底层 C++ 源码来规避常见的机器人检测。它适用于 Playwright 和 Puppeteer API，能通过 Cloudflare Turnstile、FingerprintJS 等超过 30 个检测测试，自动实现人类用户级的鼠标、键盘和滚动行为模拟。其轻松集成自动化框架如 Selenium 和 LangChain，无需其他配置即可运行，并支持持久会话以模拟真实用户行为。CloakBrowser 是网络抓取和自动化操作的高效工具，但它仅避免 CAPTCHA，并不提供直接的解决方案。
