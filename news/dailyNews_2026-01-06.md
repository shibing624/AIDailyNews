### [揭秘LLM如何选择下一个词：关注机制详解](https://kaamvaam.com/machine-learning-ai/llm-attention-explanation/)

来源：Hacker News - Newest: "llm"

发布时间：2026-01-06 14:49:39

文章详细解释了语言模型如何通过注意力机制选择词汇。以'I love you'为例，模型使用向量代表词语内在含义，利用三种过滤器（Wq、Wk、Wv）分析语境。模型通过比较“你”的查询向量与之前词汇的键，计算分值并通过Softmax权重混合创造内容向量，最终从词汇表选出最匹配词。该机制使模型能够有效预测下一个词，如选择“too”代替随机名词，这是机器学习广泛应用的基础技术。

### [BitNet：突破性 1-bit LLMs 推理框架](https://github.com/microsoft/BitNet)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-01-06 23:21:08

bitnet.cpp 是一种用于 1-bit 大型语言模型（LLMs）的官方推理框架，其优化的内核支持在 CPU 和 GPU 上进行快速无损的 1.58-bit 模型推理，未来将支持 NPU。bitnet.cpp 初始版本主要支持在 ARM 和 x86 CPUs 上进行推理，其中 ARM CPU 的加速效果为 1.37 倍到 5.07 倍，且能将能耗降低 55.4% 到 70.0%。在 x86 CPU 上，加速效果为 2.37 倍到 6.17 倍，能耗降低 71.9% 到 82.2%。该框架显著提高了大型语言模型在本地设备上的运行潜力，使得在单个 CPU 上运行100B模型的速度可与人眼阅读速度相媲美。
