### [PersonaPlex: 实现个性化语音对话的全双工模型](https://github.com/NVIDIA/personaplex)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-04-08 00:11:10

PersonaPlex 是一个实时全双工语音转换对话模型，通过文本和音频实现个性化角色控制。基于 Moshi 架构训练，它支持自然低延迟的语音交互，同时保证角色一致性。该模型包含丰富的语音集，从自然对话到更具多样化的角色音调，能够通过提示语进行角色切换。它还提供 CPU 内存不足的卸载选项以及多场景的静态和实时评估功能。文章介绍了详细的架构及使用方法，包括服务器部署、提示操作和离线评估。同时，PersonaPlex 在生成超出分布范围的对话中具备较强的泛化能力，适合探索新场景与角色设定。

### [TimesFM: 谷歌推出的时间序列预测模型](https://github.com/google-research/timesfm)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-04-08 00:11:13

TimesFM 是谷歌研究团队开发的时间序列基础模型，用于实现高精度的时间序列预测。最新版本 TimesFM 2.5 减少了参数数量，同时显著扩展上下文长度，并增加了连续分位数预测功能。除了模型升级，还重新加入了协变量支持并优化推理 API，以及针对 Flax 实现的支持。这些改进旨在提升预测能力和API使用便利性。除此之外，文档还提供了安装指南及不同后台的选择方式，支持 PyTorch 和 Jax 等框架。此项目吸引了数据科学领域的开发者，对时间序列中的高效性及灵活性有很大促进作用，适合应用于财务预测、库存优化等实践场景。文章的专业性强，信息详实，能够吸引关注预测模型优化的技术爱好者。
