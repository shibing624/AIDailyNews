### [FSM-agent-flow：自主调试的工作流引擎](https://newjerseystyle.github.io/FSM-agent-flow/)

来源：Hacker News - Newest: "llm"

发布时间：2026-02-19 23:12:17

这篇文章介绍了一种名为 FSM-agent-flow 的工作流框架，专注于自动化 LLM（大语言模型）工作流的自我验证。每个状态在前进之前都会进行验证检查，失败的状态可通过回馈机制进行重试，像是为代理管道提供测试驱动开发（TDD）解决方案。该框架支持多种分支和循环条件，无需特定运行时依赖，可适配 OpenAI 和 LiteLLM。用户可通过浏览器编辑节点，以 Python 或 JSON 导出。此工具简化了工作流的管理和实现，是一种便捷且灵活的解决方案。

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-02-19 23:48:00

Zvec 是一个开源的嵌入式向量数据库，因其高效快速而受到关注。基于阿里巴巴 Proxima 引擎，Zvec 旨在为内嵌应用提供生产级、低延迟的相似性搜索，支持稠密与稀疏向量的混合搜索。通过极简安装，用户能在几秒钟内部署并开始搜寻，无需服务器与复杂配置。作为进程内库，Zvec 可在各种环境中运行，例如笔记本、服务器甚至边缘设备。该项目在保持高性能的同时，确保了良好的社区互动与支持，为开发者提供广泛的贡献与改进机会。
