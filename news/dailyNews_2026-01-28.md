来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-01-28 23:42:13

PageIndex提供了一种无向量数据库、无分块处理及人类般文档检索的解决方案。利用类人推理，PageIndex从复杂长文档中构建语义树结构，使得大型语言模型（LLM）可以在该索引上进行推理，实现相关性而非相似性检索。新开发的PageIndex OCR优化了文档的全局结构保留，超越了许多现有的OCR工具，如Mistral和Contextual AI。PageIndex的推理RAG系统在FinanceBench基准中实现了卓越性能，展示了其特别适合于金融报告分析的特点。

### [突破性RAG开发框架：UltraRAG](https://github.com/OpenBMB/UltraRAG)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-01-28 23:42:13

UltraRAG是首个基于MCP架构设计的轻量级RAG开发框架，由清华大学、东北大学和OpenBMB联合推出。其独特之处在于通过低代码方案简化复杂工作流的管理，通过Yaml配置实现复杂的RAG逻辑。同时，UltraRAG引入了一个强大的可视化RAG开发环境，提供从管道设计到交互式对话系统的全生命周期支持，大幅提升研究效率。此外，支持模块扩展和复制，方便快速原型生成和即时交付解决方案。

发布时间：2026-01-28 23:42:11
