### [LangExtract：从未结构化文本中提取信息的利器](https://github.com/google/langextract)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-01-17 23:18:06

LangExtract是一个Python库，旨在从未结构化文本中提取结构化信息，特别适用于处理像临床笔记这样的长文档。它能够精准追踪每个提取条目的源位置，并保持输出结果的一致性。支持多种LLM模型，包括云端的Gemini系列和本地的Ollama接口。其灵活性使其适用于多种领域，支持自定义模型提供者无需对模型进行微调。LangExtract提供交互式的可视化工具以审阅提取的实体，适合需要精准和大规模处理文本数据的用户。

### [TARS：多模态 AI 代理栈](https://github.com/bytedance/UI-TARS-desktop)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-01-17 23:18:09

TARS 是一个多模态 AI 代理栈，包括 Agent TARS 和 UI-TARS-desktop 两个项目。它旨在通过将GUI代理和视觉功能引入终端、计算机和浏览器等环境，实现近似人类任务完成的工作流程。Agent TARS 提供了 CLI 和 Web UI 接口，强调多模态 LLMs 与现实工具的无缝结合。最新版本包括对多工具流控制、实时数据流和调试查看器的支持。与此同时，UI-TARS Desktop 为本地计算机提供基于UI-TARS模型的本地 GUI 代理。新功能如远程计算机操作提升了工具的实用性和智能化水平。

### [VoxCPM：真实感语音合成与语音克隆新纪元](https://github.com/OpenBMB/VoxCPM)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-01-17 23:18:06

VoxCPM是一个无标记的文本到语音系统，突出了语境感知和逼真的零样本语音克隆能力。该系统通过建模连续语音空间克服了离散标记化的局限，内置了MiniCPM-4框架，支持高效的LoRA微调，适合创建定制化语音版本。VoxCPM凭借其在中文和英文语料上训练的语境感知能力，实现了自适应语音风格生成，生成的语音具有高表现力和自然流畅性。尽管具有优势，VoxCPM存在语音克隆被滥用的风险，需要使用者谨慎使用。
