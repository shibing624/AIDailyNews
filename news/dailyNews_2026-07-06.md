### [Strix: 开源的AI渗透测试工具](https://github.com/usestrix/strix)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-07-06 00:08:06

Strix 是一种开源的 AI 渗透测试工具，专为开发者和安全团队设计，能够自动检测和修复应用漏洞。该工具采用了动态代码分析，并使用多代理协作以模拟真实黑客行为。它可以验证漏洞，生成修复补丁和合规报告，同时与 GitHub Actions 和 CI/CD 管道无缝集成，保证代码进入生产环境前的安全性。Strix 支持广泛的漏洞类型扫描，例如 SQL 注入、XSS、SSRF 等，还可自动生成漏洞证明和修复建议。其核心特点包括全套渗透测试工具、多代理协调和持续学习能力，适合快速安全测试和漏洞阻断的场景。

### [AI Berkshire：AI时代的价值投资框架](https://github.com/xbtlin/ai-berkshire)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-07-06 00:08:06

AI Berkshire 是一个集合段永平、巴菲特、芒格和李录的价值投资方法，由 AI 实现自动化投资研究的框架。它通过多Agent协作机制，提供专业级的投资分析和决策支持，并解决了普通AI无法提供明确结论和优质分析的问题。AI Berkshire的功能包括强制性立论、四大师视角对抗、反偏见机制和精确金融数据处理等，可生成结构化、高质量的投研报告。其特点是分析深度、视角多样化以及基于明确逻辑的投资建议，适合专业投资研究团队以及个人投资者。
