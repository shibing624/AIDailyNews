### [LangExtract：高效Python库文本信息提取](https://github.com/google/langextract)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-02-11 00:07:07

LangExtract是一个Python库，专注于从非结构化文本中提取结构化信息，支持定义提取任务并利用少量示例驱动模型行为。它的创新在于提供可靠的结构化输出，尤其适用于长文档提取，能够实现视觉化的交互式结果验证和跟踪。用户可以使用云端和本地大语言模型进行处理，并支持自定义模型提供者。LangExtract独特之处在于就地映射并高效处理大文档，同时支持多语言环境和多种领域应用，尤其在医学文本提取、文档结构化上表现卓越。

来源：Trending repositories on GitHub this week · GitHub
