### [Pixelle-Video：全自动化AI短视频生成利器](https://github.com/AIDC-AI/Pixelle-Video)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-05-06 00:25:03

Pixelle-Video 是一款 AI 驱动的创新短视频生成工具，通过输入一个主题即可实现文案撰写、图片生成、语音合成、背景音乐添加及视频自动合成等全套流程，完全适合零剪辑经验人士使用。支持多种视觉风格模板，自定义素材以及不同大小的视频比例，同时采用模块化设计，允许用户灵活配置每个处理环节。兼容多种 AI 模型如 GPT 和 LLM，并支持包括中文在内的多语言语音合成。适合新媒体达人、内容创作者需要高效生成高质量创意短视频的场景。

### [Scrapling：适应现代网络的轻量级爬虫框架](https://github.com/D4Vinci/Scrapling)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-05-06 00:25:03

Scrapling 是一个现代化网络爬虫框架，支持适应不同网页变化的智能解析技术。主要功能包括爬行框架（支持并发、暂停与恢复）、动态加载解析、高级反爬虫工具及代理轮换集成等，可快速抓取现代网页数据。Scrapling 也支持生成自适应的选择器，对页面动态元素具有重新定位的能力。此外，它还内置支持 TTS/TTS 和 JSON/JSONL 数据输出，提供高效的多任务和资源管理功能，非常适合高级开发者和数据爬取任务的初学者使用。
