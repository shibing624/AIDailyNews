### [Chandra OCR 2：先进文档智能技术](https://github.com/datalab-to/chandra)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-03-27 00:12:28

Chandra OCR 2 是一款先进的 OCR 模型，能够将图像和 PDF 文件转化为结构化的 HTML、Markdown 或 JSON，同时保留复杂布局信息。其支持90多种语言，使得多语言性能显著提升，对手写文档、表格及数学公式的处理表现尤为出色。最新版本优化了多语言基准、数学和表格精度，并通过两种推理模式（本地 HuggingFace 和远程 vLLM）提升了处理效率。用户可以通过 CLI 或 Streamlit Web 应用轻松使用该工具，快速完成单页或批处理任务。此外，该工具对商业使用提供灵活定价选项，是研究与文档处理领域的理想解决方案。

### [深入探索 AI 社区动态：/last30days 技能](https://github.com/mvanhorn/last30days-skill)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-03-27 00:12:28

/last30days 是一款强大的社区研究工具，帮助用户跟踪和分析过去30天内的热点话题。该技能支持 Reddit、Twitter (X)、Bluesky、YouTube 等平台，通过收集社区讨论、点分享权重、高频术语等数据，生成可靠的多来源叙述和分析，其中包括 Seedance 2.0 使用提示、新型 Nano Banana Pro 技术等内容。在最新版本 v2.9.5 中引入了诸如 Bluesky 社交平台支持、对比分析模式和项目配置文件等功能，为用户提供更丰富的数据探索方式。此外，它优化了自动保存、评论排序、多信号评分和跨市场预测等功能，为研究 AI 提供了全方位、深入的支持。工具不仅可用于 AI 提示技术研究，还适合追踪文化、新闻、音乐等热门趋势。

### [Project N.O.M.A.D.：离线知识与教育服务器](https://github.com/Crosstalk-Solutions/project-nomad)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-03-27 00:12:31

Project N.O.M.A.D. 是一个完全离线的知识与教育服务器，设计用于提供教育资源、AI、离线地图及数据工具等功能，无需持续联网即可使用。该项目可安装在基于 Debian 的操作系统，如 Ubuntu，通过浏览器访问其功能模块。特性包括离线版维基百科、Khan Academy 教程、地方地图、数据加密与分析工具等，并可通过 Docker 或本地安装方式运行。支持配置 AI 本地聊天服务和用户界面，硬件需求可根据功能模块具体调整，非常适合注重数据隐私的用户。

### [DeerFlow 2.0：超级代理执行工具](https://github.com/bytedance/deer-flow)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-03-27 00:12:31

DeerFlow 是一个开源的超级代理执行工具，可协调子代理、内存以及沙盒环境完成多功能复杂任务。其 2.0 版本进行了全面重构，提供模块化的结构，在 LangGraph 和 LangChain 上构建，内置文件系统、技能、沙盒执行和分步骤任务规划与执行功能。支持 Docker 与本地开发方式运行，同时集成智能搜索与爬虫工具 InfoQuest。该项目适用于从深度研究到超级工具管理场景，允许用户利用其扩展技能模块进行文档生成、数据处理和自动化工作流程等任务。
