### [Meta的自适应排名模型：实现LLM级广告推荐扩展](https://engineering.fb.com/2026/03/31/ml-applications/meta-adaptive-ranking-model-bending-the-inference-scaling-curve-to-serve-llm-scale-models-for-ads/)

来源：Engineering at Meta

发布时间：2026-04-01 00:00:17

Meta致力于率先采用突破性的AI推荐系统（RecSys）技术，提升用户体验并为广告主带来更好的效果。此次，Meta正在将广告推荐的运行时模型扩展到大语言模型（LLM）级别，以加深对用户兴趣和意图的理解。通过这种扩展，Meta不仅能够处理更复杂的数据，还能更精确地匹配人与内容，从而优化广告效果。相关技术解析涵盖了自适应排名模型，以及如何在推理规模曲线上实现更高效的扩展以服务于此类模型。这篇文章重点展示了Meta在机器学习模型优化和大规模系统部署方面的新突破。

### [VibeVoice: 开源语音 AI 创新](https://github.com/microsoft/VibeVoice)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-04-01 00:06:20

VibeVoice 是微软推出的开源语音 AI 框架，整合集 ASR（自动语音识别）和 TTS（文字转语音）两大核心模型，适用于长音频的高效处理。ASR 模型支持60分钟单次语音输入，并能生成带有发言人和时间戳的结构化转录结果；TTS 模型支持多语言及长达90分钟的多发言人语音生成。其创新包括超低帧率语音编码器及结合 LLM 的文本情境理解，显著提高计算效率和音频保真度。同时支持用户设定热词提升领域内容识别效果。VibeVoice 的未来社区应用潜力巨大，支持讨论、多人会话和跨语言对话等。

### [VibeVoice：领先的开源语音 AI 全能工具](https://github.com/microsoft/VibeVoice)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-04-01 00:06:26

VibeVoice 是微软推出的开源语音 AI 项目，包括语音识别（ASR）和语音合成（TTS）模型。ASR 支持 60 分钟长音频的单次转录，并提供用户自定义关键词提升识别精度；TTS 支持 90 分钟长文本生成并支持多语种与多角色对话。最新模型还引入 vLLM 推理，可大幅提升推理速度。优势包括流畅的多角色转换、实时 TTS 支持和长文本语义上下文理解。VibeVoice 旨在推动语音技术研究的协作发展，对长形式播客、对话生成等十分理想。项目附有详细的调优文档与 Colab 演示，适合开发者和研究者使用。
