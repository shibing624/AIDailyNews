### [Kronos：金融市场语言的基础模型](https://github.com/shiyu-coder/Kronos)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-05-28 01:58:14

Kronos是首个专为金融市场设计的开源基础模型，专注处理高噪声的K线数据（OHLCV）。它采用两阶段方法：将连续数据量化为离散化层次Token，然后以大型自回归Transformer进行预训练。该模型支持多种计算需求，适合预测和量化研究，并提供在线演示和详细的预测例程。进一步地，用户可以利用Qlib工具对模型进行微调，适配特定市场数据。其对金融量化研究领域的重要性显著，尤其适合量化从业者实现模型的个性化定制和优化。
