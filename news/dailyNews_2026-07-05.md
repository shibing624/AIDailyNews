### [Strix: 开放源码的AI渗透测试工具](https://github.com/usestrix/strix)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-07-05 00:04:23

Strix 是一个开源的AI驱动渗透测试工具，它提供了全套的功能集，涵盖从侦察到漏洞利用和验证。通过整合GitHub Actions和CI/CD流水线，Strix可以对每个Pull Request进行自动漏洞扫描，并在不安全代码进入生产环境前阻止其上线。它的关键功能包括自动修复、漏洞验证、漏洞补丁生成及合规测试报告等。Strix适用于应用安全测试、快速渗透测试、漏洞赏金计划，以及CI/CD管道的持续集成。该工具支持分别负责侦查、利用和后期利用的多代理操作，可实现分布式和高效全面的安全测试，特别适合开发人员和安全团队使用。

来源：Trending repositories on GitHub this week · GitHub

Strix是一款开源的AI渗透测试工具，提供了完整的渗透测试功能，包括侦查、漏洞验证和自动代码修复。其核心亮点是通过多代理AI协作进行渗透测试，并能够生成实际漏洞的概念验证（PoC）。工具支持直接集成到CI/CD流水线中，保证代码在部署前进行安全核查。它利用先进的技术，如动态代码分析、业务逻辑漏洞检测以及自动生成补丁，可有效提高测试效率并减少人工操作。适合开发者和安全团队对快速精准的安全测试需求，支持GitHub Actions轻量化运行。

### [LingBot-Map: 高效流式3D重建模型](https://github.com/Robbyant/lingbot-map)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-07-05 00:04:26

LingBot-Map是一种流式的3D几何重建模式，基于Geometric Context Transformer框架，能够在长序列场景中实现高效且稳定的实时推理。它结合坐标定位、长距离漂移校正以及动态内存存储，以高性能达到每秒约20帧的重建速度。该模型在多项基准测试中表现优异，优于现有的流式与迭代优化方法，并支持大规模场景长序列处理。特点还包括适配多种数据集及优化工具的使用，对于工业、研究场景中的3D重建需求非常适合，是一款创新性的前沿模型。
