### [微软推出开源语音 AI 模型框架：VibeVoice](https://github.com/microsoft/VibeVoice)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-04-02 00:07:17

VibeVoice 是微软推出的开源语音 AI 模型框架，支持文本到语音（TTS）和自动语音识别（ASR），采用创新的连续语音分词器与低帧率处理技术，结合大语言模型（LLM）生成高保真度语音音频。ASR 模型支持60分钟长音频单次处理，提供语者、时间戳和内容的富文本转录功能，并支持自定义词汇；TTS 模型支持生成最多90分钟、多语种和多语者的自然对话音频。特别亮点是实时流式 TTS 支持低延迟内容生成，扩展性能明显。此次发布还包含多项技术报告和示例代码，方便技术开发者探索与部署。

### [PaddleOCR：全球领先的 OCR 工具与文档 AI 引擎](https://github.com/PaddlePaddle/PaddleOCR)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-04-02 00:07:17

PaddleOCR 是一款领先的 OCR 工具与文档 AI 引擎，能够高效解析 PDF 文档和图片为结构化数据（JSON/Markdown）。最新的 PaddleOCR v3.4.0 推出了创新的文档解析算法（PP-DocLayoutV3），在解决复杂文档场景方面表现优异，支持94.5%的识别准确度。同时，PaddleOCR 提供全方位的多语言支持、表格合并、层级标题识别等功能，适用于边缘及云部署场景。支持开发者快速构建智能文档处理解决方案，广泛应用于 RAGFlow 等技术框架。

### [VibeVoice：开源语音 AI 框架助力长时间语音应用](https://github.com/microsoft/VibeVoice)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-04-02 00:07:21

VibeVoice 是微软开源的语音 AI 项目，提供了两大核心模型：长时间语音识别 (ASR) 和多角色文本转语音 (TTS)。ASR 模型可处理 60 分钟音频并支持自定义热词和结构化转录；TTS 模型支持多语种、多角色对话和长时语音生成，适合播客、对话重现等场景。创新点包括低帧率持续语音编码器和扩散式生成框架，显著提升了长序列处理的效率。项目提供文档、演示及社区模板，对研究与工业场景的语音技术发展具有推动作用，但需要注意技术使用的责任与风险。
