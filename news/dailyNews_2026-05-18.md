Scientific Agent Skills提供了135项经过优化的科学研究技能支持多种AI代理平台，使AI代理能够处理包括生物信息学、药物设计、多组学分析、物理与天文学在内的多领域复杂任务。此库包含对多种科学数据库（PubChem、UniProt等）以及Python包（RDKit、Scanpy等）的集成，并提供全面的文档和使用实例。此外，它助力研究人员通过单一命令完成多步骤科学流程，显著提升研究效率，并支持自定义和企业扩展，非常适合需要强大科学计算支持的团队或个人。

### [CLI-Anything：将所有软件转化为代理友好型](https://github.com/HKUDS/CLI-Anything)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-05-18 00:04:48

CLI-Anything旨在将现有的软件工具转化为适配AI代理的模式，以支持未来的代理用户。本文介绍了CLI-Hub平台，用户可以浏览、安装及管理社区构建的命令行工具（CLI）。同时，它展示了AI代理如何通过这些CLI实现从生成CAD设计到影片字幕的任务，以及路径预览、轨迹循环等功能。近期更新包括QGIS和Unimol Tools CLI的整合、CLI-Hub UX优化，以及针对CLI技能文档的结构调整。此项目通过“一行命令”即可使各种软件代理化，推动开源社区将复杂软件转化为易于自动化交互的形式，适合对人工智能和自动化有需求的开发者使用。

### [CloakBrowser：突破检测的隐身版Chromium浏览器](https://github.com/CloakHQ/CloakBrowser)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-05-18 00:04:52

CloakBrowser 是一款以 Chromium 为基础，进行深度隐身优化的开源浏览器。通过在 C++ 源代码层面进行 49 项补丁修复，解决了如 Canvas、WebGL、屏幕属性、音频等检测问题，使其外观与行为都与普通浏览器一致。支持 Python 和 JavaScript 下的 Playwright/Puppeteer 替换，无需复杂配置即可在 30 秒内部署。主要特点包括自动更新、零配置、持久化配置文件及通过多种反检测站点验证（如 Cloudflare Turnstile 和 FingerprintJS）。此外提供高级的用户交互模拟，包括人类化的鼠标曲线和输入行为，适合多种自动化使用场景。

### [AgentMemory：为AI助理增加持久记忆](https://github.com/rohitg00/agentmemory)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-05-18 00:04:52

AgentMemory 是一个为多种 AI 助手（例如 Claude、Codex、Cursor 等）提供持久记忆的工具。它捕捉并压缩用户的指令和交互行为，将有用的内容存入一个可搜索的记忆库，再在后续会话中自动注入上下文，避免重复解释问题。新版本支持 12 种钩子以及多协议兼容，并集成 BM25、向量和知识图相结合的混合搜索。提供出色的检索精度和显著的成本节省功能，对比竞争对手具有更多适配多工具和高效记忆处理等优势。能够提升 AI 的使用效率，特别适合开发者与技术团队的生产力工具链。
