### [LLMtary：自动化渗透测试的新利器](https://www.llmtary.com/)

来源：Hacker News - Newest: "llm"

发布时间：2026-04-09 00:08:29

LLMtary 是一款基于 Flutter 的桌面应用，利用大型语言模型（LLM）实现渗透测试的完全自动化，从目标侦查到漏洞验证和报告生成。它可以自主运行端口扫描、DNS 枚举等基础侦查，逐步构建上下文并执行深度技术分析。用户可以看到完整的漏洞确认过程，包括执行的真实命令、结果分析和迭代验证环节。其关键特性包括安全性措施，如危险命令的硬性屏蔽、命令审批模式等，以及发现后续漏洞组合攻击路径的能力。LLMtary 的报告生成功能极为强大，能够输出包含 CVSS 元数据、发现凭据和攻击链叙事的多种格式报告。此外，工具支持本地模型和云服务，适配多种环境，对从入门到专家级别的用户都极具吸引力。

### [SEO Machine：一站式SEO优化内容生成工具](https://github.com/TheCraigHewitt/seomachine)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-04-09 00:16:00

SEO Machine 是建立在 Claude Code 平台之上的工具，专注于生成 SEO 优化的长篇博客文章。该工具整合了多种功能，包括自定义命令、SEO 分析代理（如内容分析器和关键词映射等）、26 种营销技能（A/B 测试、Email 策略等），以及 Google Analytics 和 DataForSEO 数据集成。其工作流程涵盖从关键词研究到博客内容生成、内容优化以及性能评估，并提供最终的 SEO 评分和发布准备评估。用户不仅可以创建全新的内容，还能分析并优化现有内容。它通过上下文文件的个性化配置支持品牌语调、SEO 指南和内部链接策略，适合需要高效生成和优化内容的市场人员和企业使用。

### [Newton：面向机器人和物理仿真的 GPU 模拟引擎](https://github.com/newton-physics/newton)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-04-09 00:16:00

Newton 是一个基于 NVIDIA Warp 的 GPU 加速物理仿真引擎，专为机器人学和仿真研究而设计。它支持 OpenUSD 和 MuJoCo Warp 后端，强调 GPU 上计算权限、可微分性以及用户定义的扩展性。Newton 提供多种仿真示例，包括机器人、布料、逆运动学、多物理场和软体物理模拟。此引擎兼容多平台（Linux、Windows、macOS），支持 NVIDIA GPU 和 Python 环境，是研究人员和开发者快速迭代与扩展应用的理想选择。此外，其社区驱动开发使其具备开放生态下的持续改进潜力。

### [强化OpenAI Codex的强大工具：Oh-My-Codex(OMX)](https://github.com/Yeachan-Heo/oh-my-codex)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-04-09 00:16:03

Oh-My-Codex (OMX) 是一个为 OpenAI Codex CLI 提供的工作流层，旨在简化开发者的日常工作。OMX 的主要功能包括启动更强的 Codex 会话、支持从任务澄清到完成的一致性工作流、调用标准技能，以及通过 durable state 集成项目指导、计划和日志。此外，其推荐的默认模式包括灵活搭配并行执行或持续完成循环，帮助开发者高效处理多个任务。快速入门需要具备 Node.js 和 Codex CLI 等环境，主要适用于偏好使用 Codex 的开发者。整体设计上，OMX 更像是对现行 Codex 功能的增强，而非替代，是一款提升生产力的有力工具。
