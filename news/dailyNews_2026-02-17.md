### [Heretic：解除语言模型审查的全自动工具](https://github.com/p-e-w/heretic)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-02-17 23:53:28

Heretic 是一个用于自动解除基于 Transformer 的语言模型审查的工具，不需要耗费大量的后期训练。通过结合高级定向切除实现和 TPE 参数优化器，Heretic 可以完全自动化运行。它通过最小化拒绝次数和与原始模型的 KL 发散度来发现优质的切除参数，从而保留大部分原始模型的智能。很多用户评价 Heretic 生成的模型在处理敏感话题时表现出色，达到了高水准的解禁效果。Heretic 支持大多数稠密模型，并在使模型量化方面表现出色，适合对语言模型的智能语义内部研究。

### [Hummingbot：开源的自动化交易策略平台](https://github.com/hummingbot/hummingbot)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-02-17 23:53:28

Hummingbot 是一个开源框架，旨在帮助用户设计和部署自动化交易策略，可在多个中心化或去中心化交易所运行。过去一年，Hummingbot 用户产生了超过 340 亿美元的交易量。该项目提供了多种交易连接器，支持不同类型的交易市场，并欢迎社区贡献，以实现高频交易的民主化。通过 Docker 安装可以快速开启使用，并支持通过 Gateway 中间件与自动做市商协议交互。如果遇到问题，用户可以通过 Discord 社区和其他资源寻求帮助。
