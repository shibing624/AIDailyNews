### [Kronos：金融市场语言的开源基础模型](https://github.com/shiyu-coder/Kronos)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-07-23 00:20:29

Kronos 是首个专为金融市场K线数据设计的开源基础模型，采用解码器架构并进行了专门预训练。核心创新包括两阶段框架：首先用定制分词器将多维K线数据量化为层次化离散标记，其次通过Transformer模型进行自回归模式训练。Kronos 针对不同的计算需求提供了多个预训练模型版本，并通过 Hugging Face Hub 提供下载。其功能包括短期行情预测、并行预测和定制化微调，具体案例如基于中国A股市场的微调和回测。此外，官方提供了可视化预测结果展示的在线Demo，适用于汇率预测等任务，演示了模型的实时能力。

### [Hallmark：创造不显AI痕迹的设计能力](https://github.com/Nutlope/hallmark)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-07-23 00:20:33

Hallmark是一种专为Claude Code、Cursor和Codex设计的AI设计技能，旨在创建不带AI痕迹的独特设计产物。通过应用多达二十种不同主题的选项，结合微观结构和57个误差检测关卡，Hallmark保证每个成果均独一无二而非模板化设计。它支持四种主要操作：构建全新界面；对现有代码进行反模式评分；保持复制内容、信息架构和品牌标识，重新构建页面；提取设计DNA以实现指纹化且非模板化的页面生成。每个设计都会根据实际需求赋予不同的主题、结构并进行深度优化。输出成果为具有自主样式和注释的HTML和CSS文件，通过情境化响应和自定义功能满足创意需求。优良的技术品质和创新性使其非常适合追求高质量和特别设计的开发者。

### [Code Review Graph：优化代码审查的AI助手](https://github.com/tirth8205/code-review-graph)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-07-23 00:20:33

Code Review Graph是一款通过生成代码结构图来提升AI代码审查效率的工具。它基于Tree-sitter构建抽象语法树，追踪代码变更，生成最小读取集，避免AI需要重新解析整个代码库。在代码变动后，该工具通过SHA-256哈希检测进行增量更新，最快可在2秒内完成重新索引，大幅减少了代码审查中的时间和计算资源浪费，尤其在大型代码库中尤为显著（如支持减少82倍的标记）。此外，它支持广泛的编程语言并允许用户新增语言解析，提供了一流的多语言覆盖和解析能力，甚至支持Jupyter笔记本的解析。工具同时集成GitHub Action，可本地生成风险评估的PR审查报告，使用范围广阔。总体来说，是一款为高效代码评审设计的出色工具。
