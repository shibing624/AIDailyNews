### [深度代理：即插即用智能代理框架](https://github.com/langchain-ai/deepagents)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-03-18 00:06:13

Deep Agents是一个开箱即用的智能代理框架，旨在简化AI应用开发过程。它提供了任务规划、文件系统访问、Shell命令执行和子代理功能，并包含了智能默认设置以有效利用这些工具。无需复杂的配置，用户可以快速启动并根据需要定制工具、替换模型或优化提示。除了100%开源和与任何大型语言模型兼容的特点，它还支持LangGraph，提供多样化的流式处理和持久化功能。该项目的启发来自Claude Code，并强调在工具级别设定边界而非自我规范，保证框架的安全性。

### [DimOS：通用机器人操作系统](https://github.com/dimensionalOS/dimos)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-03-18 00:06:13

Dimensional OS是一款现代化的通用机器人操作系统，支持大多数机器人制造商，无需ROS即可安装。用户可以以Python开发物理应用，并通过自然语言与硬件交互。支持导航映射、感知、代理控制及空间记忆等功能。DimOS模块在机器人上运行，并利用标准化信息流进行通信，构建和管理多模块系统。通过多语言支持，与C++、Lua、TypeScript等编程语言高效协作。
