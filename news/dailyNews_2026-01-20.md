来源：Hacker News - Newest: "AI"

发布时间：2026-01-20 23:02:53

OpenAgents 宣布支持 Agent2Agent (A2A) 协议，实现不同 AI 代理之间的无缝连接。A2A 被比喻为 AI 代理的 HTTP，使得各种框架的代理可以轻松互通。它使用 JSON-RPC 2.0 实现标准化交流，且 OpenAgents 网络现成为一个统一的代理注册表，可通过 gRPC 与 A2A 代理自动进行协议翻译。这一协议的引入让开发者更容易构建多样化框架的研究团队，增强了系统的互操作性，并有助于解决 AI 代理生态系统的碎片化问题。

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-01-20 23:29:00

LangExtract 是一个利用大语言模型（LLMs）从非结构化文本中提取结构化信息的Python库。其优势在于精确的消息溯源和可靠的结构化输出，适配长文档，通过分块及多次提取等策略提高召回率，并且支持多种LLM平台。用户可以定义提取任务，指定示例以影响LLM对提取任务的知识利用程度。它还能生成交互式可视化文件，方便审阅提取的实体。该库适用于不同领域的文档信息提取，不需模型调优，同时提供了扩展自定义模型提供者的接口。
