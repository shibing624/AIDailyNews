### [BitNet 扩展 LLMs 的1比特推理引擎](https://github.com/microsoft/BitNet)

来源：Trending repositories on GitHub this week · GitHub

发布时间：2026-03-17 00:05:50

BitNet 绘制了一个新的低位神经网络推理框架，通过优化推理内核，支持对1.58-bit模型进行快速且无损的推理。它在ARM和x86 CPU上提供显著的加速和能效改善。此外，该框架实现了大型LLM模型可以在普通CPU上比肩人类阅读速度的推理性能，极大地拓展了在本地设备上运行大型语言模型的可能性。这些功能使得bitnet.cpp非常适合于低能耗和高效能的边缘计算任务。
