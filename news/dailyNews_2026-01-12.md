来源：Hacker News - Newest: "AI"

发布时间：2026-01-12 22:56:53

### [多平台自媒体数据采集利器 MediaCrawler](https://github.com/NanmiCoder/MediaCrawler)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-01-12 23:23:58

MediaCrawler 是一个强大且功能丰富的多平台自媒体数据采集工具，支持抓取小红书、抖音、快手、B站、微博、贴吧、知乎等平台的公开信息。该工具基于 Playwright 浏览器自动化框架，能够保留登录态，从而无需JS逆向破解复杂算法。这使得使用者可以更低的技术门槛实现数据爬取。MediaCrawlerPro 版本则增强了断点续爬功能，多账号和IP代理支持，并去除了 Playwright 依赖，优化了架构，适合构建大型项目。此外，该项目为非商业用途，开发者强调以学习研究为目的使用。本工具提供了包括 WebUI 界面，可视化编辑和爬虫参数配置等，能实时查看运行状态并导出数据。
