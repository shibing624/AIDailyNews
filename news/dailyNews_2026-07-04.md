### [Skillsaw：提高AI代码代理的技能质量工具](https://skillsaw.org/)

来源：Hacker News - Newest: "AI"

发布时间：2026-07-04 00:13:25

Skillsaw 是一个专门用于校对和改进 AI 代码代理相关文件（如 skills, plugins, CLAUDE.md 和 AGENTS.md）的工具。通过超过40条规则，Skillsaw 可以检测弱语言、矛盾点、注意力盲区以及结构问题，并实现自动修复。其特性包括：基于 deterministic 的自动修复功能（通过 ‘skillsaw fix’ 指令），支持自定义规则和 pip 安装的规则插件，并可对单个规则设置阈值。同时，它与 GitHub 和 GitLab 集成，提供内联 PR 评论、去重和自动线程解决。此外，它还带有一个详细的演示视频，展示如何从头开始对一个项目进行评分、修复和配置，适用于开发者对项目质量的严格需求。

Strix 是一个开源的 AI 渗透测试工具，其核心目的是通过模拟真实黑客行为，帮助开发者和安全团队快速有效地发现并修复应用程序漏洞。该工具具有全面的渗透测试功能，包括自动化攻击面扫描、漏洞验证（如 SQL 注入、XSS 等），并提供实际的概念验证 (PoC)。Strix 支持 GitHub Actions 和 CI/CD 集成，可在代码推送时自动扫描并阻止不安全代码进入生产环境。此外，它支持一键自动修复生成的安全补丁，简化开发者的处理过程。通过多代理编排，Strix 提供分布式测试功能，模拟红队的协作攻击流程。工具易用性高，面向开发者优先设计，且生成合规报告的能力极强。
