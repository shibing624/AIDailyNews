### [Claude魔法字符串拒绝服务的风险与防范](https://hackingthe.cloud/ai-llm/exploitation/claude_magic_string_denial_of_service/)

来源：Hacker News - Newest: "llm"

发布时间：2026-01-26 20:33:39

这篇文章讨论了Claude模型在处理“魔法字符串”时的风险，这一字符串用于触发模型的拒绝响应，但它也可能被攻击者用作制造拒绝服务攻击的手段。攻击者可以将字符串注入提示上下文，使Claude持续拒绝响应处理。为了防范此类风险，文章建议开发者需要明确处理拒绝，并在上下文中重置受到该字符串影响的部分。此技巧对提高应用的鲁棒性至关重要，特别是在处理未知或不受信任的输入时，需将其当作潜在的注入攻击面来对待。

### [PageIndex：无向量的自举RAG](https://github.com/VectifyAI/PageIndex)

来源：Trending Python repositories on GitHub today · GitHub

发布时间：2026-01-26 23:27:34

PageIndex是一种无向量、推理为基础的检索与生成（RAG）系统，通过在长文档上构建层次树索引，并利用大型语言模型（LLM）进行推理以实现类人的文档检索和分析。与传统的基于相似性的向量检索不同，PageIndex强调基于文档结构和推理的相关性检索，不需要向量数据库或分块。该系统在文档分析中表现出色，达到98.7%的FinanceBench准确率，适用于金融报告和法律文档等需要多步骤推理的专业文档。
